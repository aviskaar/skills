## Compliance Readiness Scoring

Use this scorecard monthly to assess compliance readiness:

```
COMPLIANCE READINESS SCORECARD

Dimension                          Score (0–10)  Weight  Weighted Score
──────────────────────────────────────────────────────────────────────
Control implementation             [  ]          30%     [   ]
Evidence completeness              [  ]          25%     [   ]
Finding remediation (on time)      [  ]          20%     [   ]
Policy currency (all <12 months)   [  ]          10%     [   ]
Staff training completion          [  ]          10%     [   ]
Vendor compliance status           [  ]          5%      [   ]
──────────────────────────────────────────────────────────────────────
TOTAL READINESS SCORE:                                    [   /10]

Score Interpretation:
9.0–10.0: Audit-ready; minimal risk
7.0–8.9:  Generally ready; address gaps before audit window
5.0–6.9:  Significant gaps; remediation plan required immediately
<5.0:     Not ready; do not schedule audit; escalate to CISO
```
